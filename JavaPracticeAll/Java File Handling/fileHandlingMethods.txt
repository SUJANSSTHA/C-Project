What is file Handling ? 
    File handling defines how we can read and write data on a file. Java IO package contains all the classes through which we can perform all the input and output operations in the file.

File Handling Methods
1. canRead()
2. canWrite()
3. createNewFile()
4. Delete()
5. Exists()
6. Lengths()
7. getName()
8. getAbsolutePath()
9. Mkdir()
10. List()
11. Read()
12. Write()
13. rename()
